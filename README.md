# Assessment Project

## 📌 Overview

This repository contains database design and data-related files for the assessment.

The project includes:

* ERD (3NF design)
* Kimball dimensional model
* SQL scripts
* Supporting reference files

---

## 📂 Files Description

* **ERD_3NF.html**
  Entity Relationship Diagram in Third Normal Form (3NF)

* **Kimball_style_model**
  Dimensional model based on Kimball methodology

* **measurements_mysql**
  SQL scripts for database creation and data insertion

* **pipeline_proposed_improvement**
  Proposed improvements for data pipeline

* **reference**
  Supporting documents and notes

## 🛠️ Tools Used

* MySQL Localhost
* Draw.io / ERD Tool
* GitHub
